## EN

### Requirement : 

- Vs code
- Discord js v12 (npm i discord.js@12)
- mysql (npm i mysql2)
- Fivem server
- Host + DB
- SQL ban (https://github.com/Nauxyyy/BanSQL)

### Instalation : 

- Config the config file
- node main.js
